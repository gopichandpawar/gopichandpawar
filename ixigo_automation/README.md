# Ixigo Hotel Booking Automation

## Project Overview
Automates Ixigo hotel booking flow using Selenium WebDriver and Pytest
till the payment page.

## Tech Stack
- Python
- Selenium WebDriver
- Pytest
- Chrome Browser

## Test Coverage
- City search
- Hotel booking
- Popup & window handling
- OTP verification
- Guest details
- Payment page navigation

## How to Run
pip install -r requirements.txt
pytest
