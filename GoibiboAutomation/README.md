# Goibibo Bus Booking Automation

## Project Overview
Automates Goibibo bus booking flow using Selenium WebDriver and Pytest
till the payment page.

## Tech Stack
- Python
- Selenium WebDriver
- Pytest
- Chrome Browser

## Test Coverage
- Source & destination selection
- Bus search
- Seat selection
- Passenger details
- Payment page navigation

## How to Run
pip install -r requirements.txt
pytest

## Notes
- Dummy data used
- No real payment is done
- Script is for learning purposes
